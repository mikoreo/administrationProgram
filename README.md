# administrationProgram
dit programma is gemaakt voor een school opdracht, het heeft de volgende functionaliteiten.

je start met 5 keuzes.
1: add user.
2: show user.
3: delete user.
4: edit user.
0: close program.

1: add user.
je maakt hier je user aan. om dit te kiezen type je 1 in in het keuze menu. je krijgt dan 4 vragen gesteld. wat is je naam? wat is je telefoon nummer? wat is je email adres? wat is je leeftijd? als je alles hebt ingevuld kom je terug bij het keuze menu.

2: show user.
om hier te komen moet je 2 typen in het keuze menu. je krijgt hier alle aangemaakte users te zien met een eigen ID nummer, als er nog geen users zijn aangemaakt staat er dat het leeg is. hierna kan je weer terug naar het keuze menu.

3: delete user.
om hier te komen moet je 3 typen in het keuze menu. ten eerste krijg je alle users te zien netzoals in show user. je kan hier een gekozen user verwijderen door zijn eigen ID nummer in te vullen. daarna ga je weer terug naar het keuze menu.

4: edit user.
om hier te komen moet je 4 typen in het keuze menu. je kan hier je user info editen. je kiest eerst het ID van de user die je wilt editen en daarna krijg je de 4 vragen op nieuw te zien en kan je het opnieuw invullen.
